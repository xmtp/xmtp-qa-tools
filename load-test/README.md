# XMTP Load Test

High-performance load testing for XMTP (5M+ msgs/day).

## Quick Start (Auto-tuned)

```bash
cd load-test
yarn install

# 1. Auto-tune for your system
npm run auto-tune

# 2. Run the recommended setup command it gives you

# 3. Run the auto-tuned test
npm run test:auto
```

## Manual Usage

```bash
# Setup: Create identities and groups (defaults to D14n staging)
npx tsx setup.ts -i 100 -g 10 -m 20 -e dev

# Run: Full load test (24hr)
npm run test

# Run: 1-minute burst test
npm run test:burst

# Run: Simple runner
npm run test:simple

# Clean: Remove test data
npm run clean
```

## Config

- `artillery-config.yml` - Full 24hr test
- `artillery-config-burst.yml` - 1min burst test
- `artillery-config-auto.yml` - Generated by auto-tune
